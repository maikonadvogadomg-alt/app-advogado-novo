Como criar regras de captura/exclusão

Regex (Expressões Regulares) são padrões para buscar ou excluir texto. Use nos campos "Capturar por REGEX" (para destacar trechos) ou "Excluir por REGEX" (para remover).

Comandos Básicos:

  • \b: Limite de palavra (ex.: \bFraude\b pega só "Fraude", não "Fraudez").
  • ^: Início de linha (ex.: ^Modelo exclui linhas começando com "Modelo").
  • $: Fim de linha (ex.: deferimento$ pega linhas terminando com "deferimento").
  • |: Ou (ex.: Fraude|Estelionato pega um ou outro).
  • [\\s\\S]{0,500}: Pega até 500 caracteres após o termo (útil para captura de contexto).
  • .*?: Pega qualquer texto até o próximo padrão (ex.: Contrato.*?assinatura pega de "Contrato" até "assinatura").

Exemplos para Captura (Extrator):

  • \b(Falsificação|Fraude)\b[\\s\\S]{0,500}: Pega trechos com "Falsificação" ou "Fraude" + 500 caracteres de contexto.
  • \bSentença\b[\\s\\S]*?(?=\bId\s+no\s+PJE|\bEvento\s+\d+|$): Pega trecho de "Sentença" até um identificador.

Exemplos para Exclusão (Filtro):

  • ^Modelo\s+Padrão: Exclui linhas começando com "Modelo Padrão".
  • ^(?=\s{3,}): Exclui linhas com 3 ou mais espaços no início (formatação estranha).

Dica: Teste regex em ferramentas como regex101.com antes de usar aqui.
